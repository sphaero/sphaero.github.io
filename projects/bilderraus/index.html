Bilderraus

Bilderraus was a project for the event VJ op de Dom commissioned by Vrede van Utrecht 2013.

To create content for a part of the program of VJ op de Dom I worked with Aldo Hoeben and z25 to develop some new video mapping techniques.

Subsequently, z25 invited me amongst a number of other artists to participate in creating the content of this show.

We had a theme of building up and breaking down. Each artist had a part in these chain of events resulting in a 20 minute long show. Each animation always ended and started with an animation of Angels and Devils in which one of the two prevailed following a central theme of breaking down and building up.

Through the techniques and methods that where offered by z25 to the artists we were able to create a complete 3D spatial experience using the geometry of the cathedral.

This versatile diverse visual spectacle was “Bilderraus”. My contribution starts at 5:15.
