Linie in Lichterlaaie

“Fort Blauwkapel” is part of the New Dutch Water Line in Utrecht. For the “Linie in Lichterlaaie” festival wel created a very simple distributed setup to project on all the windows of the building. We created some program logic to control Raspberry Pi devices attached to video projectors from within Blender. This was no rocket science but the simplicity of the system proofed to be very powerfull. Especially for artists wanting to play with massive projections.

We created a 3D photo collage of soldiers who used this building around the first World War.

The military barracks became a visual time machine which brought back a part of Dutch war history.

This project was realised in cooperation with the MAPLAB
